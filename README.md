# Disney Movies
Disney Movies 

### Pre-Requirement 

- NodeJs 
- MongoDB 

## Set Up Client 

### Navigate and Dependancy Install 
```
cd client 
npm install
```
### Start Client 
```
npm run start
```
Open in Browser http://localhost:3000

## Set Up Server 

### Navigate and Dependancy Install 
```
cd client 
npm install 
```
### Set up environment Variables 
```
MONGODB_URL
```
### Start Server 
```
npm run start 
```
Server Start http://localhost:3001
